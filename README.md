# GarageOpener



## Installation

```ruby
gem install bundler
```

And then execute:

    $ bundle
    
## Setup

Two files needs to be edited with your MQTT settings.
`lib/garage_opener.rb` and `views/index.erb`.

Change the default basic auth usernamne and password in `app.rb`.

## Usage

```ruby
rackup
```
